SIM E02 codStPis Código de Situação Tributário do PIS btfit821 pits C 2 X CASE length(pits) WHEN 0 then '99' ELSE pits

Caso não exista conteúdo de Situação tributária do PIS (pits), será gravado "99". Caso contrário, prevalece o que houver no campo.
SIM E03 vlItem Valor total dos itens btfit821 gamt Double 8 X Sem observações
NÃO E04 codBaseCalcCredPc Código da Base de Cálculo do Crédito btfit821 pccb C 2 X WHEN pccb is not null THEN pccb
WHEN vpis > 0 and pccb is null and opfc in ('1252','2252') THEN '04'
WHEN vpis > 0 and pccb is null and opfc in ('1101','2101') THEN '02'
ELSE null

Aqui, é validado se existe Código da Base de Cálculo do Crédito PIS/COFINS (pccb) para sua gravação. Caso não exista, mas se encontre algum Valor de PIS/PASEP (vpis), será então validado por CFOP:
- CFOPs 1252, 2252: Será gravado "04";
- CFOPs 1101, 2101: Será gravado "02"

Se nenhuma dessas condições forem satisfeitas, será gravado NULL.
